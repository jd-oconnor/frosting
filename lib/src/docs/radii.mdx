# Radii

The radii object contains the radii used for the `border-radius` property.

```ts
export const radii = {
  1: "2px",
  2: "4px",
  3: "6px",
  4: "8px",
  5: "10px",
  6: "12px",
  7: "14px",
  8: "16px",
  9: "18px",
  10: "20px",
  11: "30px",
  12: "36px",
}
```

## Usage

```ts
import { style } from "@vanilla-extract/css"
import { vars } from "../../../variables"

export const softCorner = style({
  borderRadius: vars.radii[3],
  boxSizing: "border-box",
})
```
